Variables

Ah, the first ever version of BINLang - it's as good as new! The nostalgia - it reminds me of the good old times... wait. What were we at again? Oh yes, the BINLang compiler! The way to define a variable using BINLang is to use the DEF keyword. This keyword typically uses the following structure: DEF <identifier>.

The <identifier> template represents the identifier that would get treated as a variable later on in the program. Variables can be unloaded once they don't need to be used anymore.

This is how to define a variable:

DEF variable
DEF AppleBanana
DEF camelCase

This is how NOT to define a variable:

DEF vari4ble123
DEF
DEF h&lloTh%r#

If you learnt how to define a variable by looking at the examples, then good for you! You did an excellent job.
